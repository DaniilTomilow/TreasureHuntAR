#  How to install Node.js Stuff

### Notice the order!
```
npm install express
npm install body-parser
npm install multer
npm install sqlite3

```

### start the server just tap in linux
```
node server.js
```

### if the database exists
```
rm treasureHuntAR.db
```

### combine the command with pipe
```
rm treasureHuntAR.db | node server.js
```
